Common css properties

Table Properties

Property Name Description
display: Display You can choose to play inline or block or hide and element.
margin: Margin Is the outer box of box elements, is the space between elements.
align: Align [text, items, etc.] With this you can align text, elements, etc. in the center, justify, etc.
font-family: Font family With this you can change the font like any other text editor.
padding: Padding Yes, like the bear. Is the space between an element and the margin.
background-color: Background color With this you can fill out any box, like this.
color: Foreground color With this property you can color any text or element.
opacity: Opacity You can modify the opacity of the element or the background-color. 0 is transparent .5 = 50% and 1 is 100%.
box-sizing: Box sizing This property allows you to make the box auto sizing.
width: Width Is the total width of the element.
height: Height Is the total height of the element.
